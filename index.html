En las distintas etiquetas existen algunas como:

Tabla de Etiquetas HTML

Etiqueta Descripción Ejemplo
<h1> ... </h1> Define el encabezado principal de la página. Existen seis niveles de encabezados (<h1> a <h6>), siendo <h1> el más importante. <h1>Este es un Encabezado</h1>
<p> ... </p> Define un párrafo de texto. <p>Este es un párrafo.</p>
<a> ... </a> Define un enlace o hipervínculo. <a href="https://www.example.com">Visitar sitio</a>
<img> Incrusta una imagen en la página. <img src="imagen.jpg" alt="Descripción de la imagen">
<ul> ... </ul> y <li> ... </li> Define una lista desordenada (<ul>) y cada ítem de la lista (<li>). <ul><li>Item 1</li><li>Item 2</li></ul>
<div> ... </div> Define un contenedor de bloque para agrupar contenido. <div>Contenido aquí</div>
<span> ... </span> Define un contenedor en línea para agrupar contenido. <span>Texto en línea</span>
<table> ... </table> Define una tabla. <table>...</table>
<tr> ... </tr> Define una fila en una tabla. <tr><td>Celda 1</td><td>Celda 2</td></tr>
<td> ... </td> Define una celda en una tabla. <td>Contenido de la celda</td>
<form> ... </form> Define un formulario HTML. <form>...</form>

Existen varias como :

  • Head
  • Title
  • Body
  • Footer
  • h1 hasta h6
